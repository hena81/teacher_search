# نظام توزيع الملاحظين على لجان الاختبارات

نظام ويب لإدارة توزيع المعلمين على لجان اختبارات الفترة الدراسية.

## المتطلبات

- Python 3.9 أو أحدث
- pip

## التثبيت والتشغيل

### 1. إنشاء بيئة افتراضية (اختياري لكن مستحسن)

```bash
python -m venv venv

# على Windows:
venv\Scripts\activate

# على Linux/Mac:
source venv/bin/activate
```

### 2. تثبيت المتطلبات

```bash
pip install -r requirements.txt
```

### 3. تشغيل التطبيق

```bash
python app.py
```

سيعمل التطبيق على: http://127.0.0.1:5000

## الصفحات

| الصفحة | الرابط | الوصف |
|--------|--------|-------|
| البحث | `/search` | البحث عن توزيع معلم بالرقم المدني |
| الرفع | `/upload` | رفع ملف Excel وعرض/إدارة البيانات |

## صيغة ملف Excel

يجب أن يحتوي ملف Excel على الأعمدة التالية:

| اسم العمود |
|------------|
| الرقم المدني |
| الاسم |
| القسم |
| التكليف |
| اللجنة الرئيسية |
| اللجنة الفرعية |
| موقع اللجنة |
| اليوم |
| التاريخ |

## API Endpoints

| Method | Endpoint | الوصف |
|--------|----------|-------|
| GET/POST | `/search` | صفحة البحث |
| GET | `/api/search?civil=...` | البحث JSON API |
| GET/POST | `/upload` | صفحة الرفع |
| GET | `/api/assignments` | جلب كل السجلات JSON |
| POST | `/api/delete_all` | مسح كل البيانات |
| GET | `/api/template` | تحميل قالب Excel |

## ملاحظات

- قاعدة البيانات (`data.db`) تُنشأ تلقائياً عند أول تشغيل
- الحد الأقصى لحجم الملف: 5 ميجابايت
- يتم تحديث السجل إذا كان الرقم المدني موجوداً مسبقاً
